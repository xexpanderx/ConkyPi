# ConkyPi
ConkyPi for Raspberry Pi4 and external monitor with 900x1440 resolutiojn (vertical)

<b>Requirements</b>

- Overpass font
- pyowm
- w3m
- openweather api-key
- Monitor with 900x1440 resolution

<b>Install</b>

`sh install.sh`

<b>Start conky inside ~/.conky/ConkyPi</b>

`sh start_conky.sh`
